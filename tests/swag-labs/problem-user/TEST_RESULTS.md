## problem_userで確認された問題点まとめ

problem_userでは以下の問題が意図的に発生するように設計されています：

1. **ソート機能の不具合**: 名前降順、価格昇順、価格降順のソートが正しく動作しない
2. **Last Nameフィールドの入力不可**: チェックアウト画面でLast Nameフィールドに入力できない
3. **商品削除後のボタン状態不整合**: 一覧画面でRemoveボタンをクリックしてもAdd to cartボタンに戻らない
4. **商品詳細画面でのカート追加不具合**: Add to cartボタンをクリックしてもRemoveボタンに変わらない

これらの問題により、problem_userでは正常な購入フローを完了することができません。
